### Troubleshooting

If tests fail to execute automatically:
1. Verify `playwright.config.ts` has Chromium project configured
2. Check `baseURL` is set to OrangeHRM demo site
3. Ensure test credentials are correct in `.env`
4. Run `npx playwright install` to verify browser installation
